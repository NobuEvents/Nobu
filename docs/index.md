# Nobu Streaming Platform

Welcome to the documentation for **Nobu**, a high-performance, real-time event streaming and data integration platform.

## Features

- **Real-time Event Routing**: Low latency data moving.
- **CDC Support**: Capture changes from databases like SAP HANA.
- **Lakehouse Native**: Stream directly into Apache Iceberg.
- **Modular Architecture**: Built on Quarkus and LMAX Disruptor.
